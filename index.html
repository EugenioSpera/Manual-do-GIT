Manual do Git

O que é o GIT

O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux. Um número impressionante de projetos de software depende do Git para controle de versão, incluindo projetos comerciais e de código-fonte aberto.

Como iniciar o GIT

Para iniciar um repositório usamos o comando git init

Como adicionar o nome e email de quem esta programando

  • Usamos o comando git config --global user.name e o nome de quem esta programando
  • Usamos o comando git config --global user.email mais o email de quem esta programando para demonstrar o email ligado ao GIT HUB

Como linkamos o git local ao Git remoto

Usando o comando git remote add origin + endereço do GIT HUB podemos ligar o nosso repositório local ao repositório remoto, fazendo com que o repositório remoto passe a acompanhar tudo o que esta sendo feito.

Como adicionar um arquivo ao repositório

  • Para adicionarmos apenas um arquivo, usamos o comando git add mais o nome do arquivo
  • Para adicionarmos todos os arquivos de uma pasta, usamos o comando git add .
.

Comitando um arquivo ao repositório

Para comitar um arquivo ao nosso repositório remoto usamos o comando git commit -m"Mensagem que queremos para saber o que estamos fazendo" Fazendo isso criamos um ponto de restauração.

Podemos também mudar a mensagem colocado no comando commit usando git commit --amend -m

Adicionando arquivo ao repositório remoto.

Usando o comando git push origin mais o repositorio, adicionamos um arquivo ou pasta do repositório local para o repositório remoto

Comando Git remote

    Podemos usar o comando git remote --v para verificar qual repositório esta sendo usado no momento Também pode ser usado o comando git remote rm origin para deslinkar o repositório que esta sendo usado Outro comando utilizado é o git remote rename origin, que serve para renomear o nome do repositório que esta sendo utilizado

Comitando e adicionado ao mesmo tempo

Podemos usar o comando git commit -a -m"mensagem que você quer"

Git Log

  • Usamos o comando git log para verificar tudo o que foi comitado até esse momento
  • Usando o comando git log --author e o nome que voê quer pesquisar, ele mostra todos os commits feitos por esse programador.
  • Para pesquisarmos em toda a extensão dos comando alguma palavra específica, usamos o comando git log --grep mais a palavra a ser buscada.
  • Outro comando utilizado é o git short log, que nos mostra um log reduzido dos commits.
